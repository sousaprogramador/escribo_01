## Rodar a aplicação

#### Crie os containers com Docker

```bash
$ docker-compose up
```

#### Accesse no browser

```
http://localhost:8000
```

### Enviar um POST para  

```
http://localhost:8000/api/desafio

```

### HEADERS
```
Accept : application/json

Content-Type : application/json
```

### BODY 
```
{
    "value":"13"
}
```
